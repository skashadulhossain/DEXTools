  • Start using keyboard navigation mode. Press Alt + Z
  • Quick search. Press Ctrl + K
  • Change symbol. Start typing symbol name
  • Change interval. Press number or comma
  • Indicators, Metrics and Strategies. Press slash
  • Load layout. Press period
